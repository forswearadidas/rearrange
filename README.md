# **Rearrange**
This module is for rearranging the given names.

Turn "LastName, FirstName" to "FirstName, LastName".

# Example
* calling `rearrange_name("Turing, Alan")` >> `"Alan Turing"`
* calling `rearrange_name("Hopper, Grame M.")` >> `"Grame M. Hopper"`
* calling `rearrange_name("Voltaire")` >> `"Voltaire"`
